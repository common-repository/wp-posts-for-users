== Description ==
Posts and products for users, es un plugin para WordPress que permite mostrar una categoría de artículos (posts) o productos a usuarios registrados a través  del uso de un shortcode. Este plugin hace uso de algunos recursos de bootstrap 4.3.0.

= Lista de shortcodes disponibles en plugin =
Posts for users, ¿Como usar el shortcode para artículos o entradas?

// Muestra 4 entradas de todas las categorías
[wppfu_posts_for_users post_type=&quot;post&quot; post_number=&quot;4&quot;] 

// Cambia la etiqueta del botón Leer más
[wppfu_posts_for_users post_type=&quot;post&quot; label_btn=&quot;Ver más&quot;] 

// Muestra 2 columnas de entradas (solo desktop 2, 3 y 4 columnas)
[wppfu_posts_for_users post_type=&quot;post&quot; post_col=&quot;2&quot;] 

// order_post=&quot;DESC&quot; ordena los post ascendente o descendente
[wppfu_posts_for_users post_type=&quot;post&quot; post_id=&quot;ID&quot; post_number=&quot;NUMBER&quot; order_post=&quot;DESC&quot;] 

// excerpt_post=&quot;15&quot; es el número de palabras para mostrar (longitud del extracto)
[wppfu_posts_for_users post_type=&quot;post&quot; post_id=&quot;ID&quot; post_number=&quot;NUMBER&quot; excerpt_post=&quot;15&quot;] 

// msg_user=&quot;TEXT&quot; muestra un mensaje a los usuarios registrados si no hay entradas
[wppfu_posts_for_users post_type=&quot;post&quot; post_id=&quot;ID&quot; post_number=&quot;NUMBER&quot; msg_user=&quot;TEXT&quot;] 

// msg_non_user=&quot;TEXT&quot; muestra un mensaje a los usuarios no registrados
[wppfu_posts_for_users post_type=&quot;post&quot; post_id=&quot;ID&quot; post_number=&quot;NUMBER&quot; msg_non_user=&quot;TEXT&quot;] 

Products for users, ¿Como usar el shortcode para productos?

// Muestra 4 productos de todas las categorías
[wppfu_posts_for_users post_type=&quot;product&quot; post_number=&quot;4&quot;] 

// Cambia la etiqueta del botón Leer más
[wppfu_posts_for_users post_type=&quot;product&quot; label_btn=&quot;Comprar&quot;] 

// Muestra 2 columnas de productos (solo desktop 2, 3 y 4 columnas)
[wppfu_posts_for_users post_type=&quot;product&quot; post_col=&quot;2&quot;] 

 // order_products=&quot;rand&quot; ordena los productos aleatoriamente
[wppfu_posts_for_users post_type=&quot;product&quot; post_products=&quot;slug&quot; post_number=&quot;NUMBER&quot; order_products=&quot;rand&quot;]

// excerpt_post=&quot;15&quot; es el número de palabras para mostrar (longitud del extracto)
[wppfu_posts_for_users post_type=&quot;product&quot; post_products=&quot;slug&quot; post_number=&quot;NUMBER&quot; excerpt_post=&quot;15&quot;] 

// msg_user=&quot;TEXT&quot; muestra un mensaje a los usuarios registrados si no hay entradas
[wppfu_posts_for_users post_type=&quot;product&quot; post_products=&quot;slug&quot; post_number=&quot;NUMBER&quot; msg_user=&quot;TEXT&quot;] 

// msg_non_user=&quot;TEXT&quot; muestra un mensaje a los usuarios no registrados
[wppfu_posts_for_users post_type=&quot;product&quot; post_products=&quot;slug&quot; post_number=&quot;NUMBER&quot; msg_non_user=&quot;TEXT&quot;]

== Installation ==
Instalación desde el repositorio oficial de WordPress:
1. Escribe Daniel Esparza en la barra de búsqueda.
2. Selecciona el plugin para instalar.
3. Activa el plugin.
4. Lee las instrucciones de uso en la sección '<> DE Plugins' dentro del menú de administración de WordPress.
5. ¡Disfruta!

Instalación manual (con archivo zip):
1. Escribe Daniel Esparza en la barra de búsqueda del repositorio oficial de WordPress.
2. Descarga el plugin.
3. Sube el plugin desde el panel de administración de WordPress, Plugins -> Añadir nuevo -> Subir plugin -> Seleccionar archivo (buscar el archivo en nuestro equipo) -> Instalar ahora.
4. Activa el plugin desde el menú 'Plugins' de WordPress.
5. Lee las instrucciones de uso en la sección '<> DE Plugins' dentro del menú de administración de WordPress.
6. ¡Disfruta!
